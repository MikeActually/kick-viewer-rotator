# Kick Stream Rotator for Rust

Chrome extension to rotate through Kick streamers as long as they are streaming Rust. You can add streamers and set a maximum number of hours to watch per streamer.

## How to use

1. Download and extract zip from [latest release page](https://github.com/MikeActually/kick-viewer-rotator/releases/latest)
1. Go to Chrome
1. Access "Manage Extensions"
1. Enable developer mode
1. Click "Load Unpacked"
1. Select directory including the files you downloaded
1. Enable the Extension
1. Click on details
1. Click on "Pin app to toolbar"
1. Open the extension from the Toolbar icon
1. Click "setup" button
1. Add desired channels
1. Click "Show Status" button
1. When ready to watch, click "Start"
